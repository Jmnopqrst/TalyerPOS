# Seller Deployment Checklist

Use this checklist when selling or installing TalyerPOS for a new customer.

## 1. Before Delivery

- confirm customer shop name
- confirm workstation specs
- confirm Windows version
- confirm receipt printer model
- confirm QR scanner model
- confirm backup storage option
- prepare installer
- prepare license/trial terms
- prepare support contact details

## 2. Installation Day

- install TalyerPOS
- configure database path if using custom per-store database
- launch app
- confirm fresh database creation
- change default passwords
- configure business identity
- configure receipt settings
- configure payment methods
- add services
- add inventory categories
- add suppliers
- add starting inventory
- add staff and users
- add mechanics
- configure payroll if included
- print mechanic QR ID cards
- configure backup folder
- create first backup

## 3. Hardware Verification

Receipt printer:

- printer appears in Windows
- test print works
- TalyerPOS receipt output works

QR scanner:

- scanner types QR value into Notepad
- scanner sends Enter after scan
- TalyerPOS global attendance scan works
- Time In and Time Out both work

## 4. Training Checklist

Train staff on:

- login
- POS checkout
- void/refund approval
- job order creation
- job completion and payment
- inventory stock-in
- inventory adjustment
- daily reports
- backup creation
- QR attendance scan

Train owner/admin on:

- users
- settings
- audit logs
- payroll workflow
- restore preview
- database reset risks

## 5. Acceptance Test

Before leaving:

- create one test sale
- create one test job order
- complete and pay test job
- scan mechanic QR for Time In
- scan mechanic QR for Time Out
- generate test report
- create backup
- confirm backup file exists

## 6. Handover Notes

Give customer:

- login credentials
- backup folder location
- support contact
- quick user guide
- note that each shop has a separate database
- warning not to delete database or backup files

## 7. Post-Install Follow-Up

Within 1 week:

- verify backups are running
- check user questions
- review low stock setup
- confirm receipt formatting
- confirm payroll workflow if enabled

# Installation and Fresh Start Guide

This guide is for installing TalyerPOS in a new shop or workstation.

## 1. System Requirements

Recommended workstation:

- Windows 10 or Windows 11
- 64-bit processor
- 4 GB RAM minimum, 8 GB recommended
- 1 GB free disk space minimum
- receipt printer if printing receipts
- USB or Bluetooth keyboard-wedge QR scanner for mechanic attendance
- backup storage location, preferably external drive or cloud-synced folder

## 2. Install TalyerPOS

Use the installer generated by:

```bash
npm run dist
```

Installer output is created under:

```txt
release/
```

Default installer name:

```txt
TalyerPOS-Setup-<version>.exe
```

Installation steps:

1. Run the installer.
2. Choose install location.
3. Allow desktop/start menu shortcuts.
4. Launch TalyerPOS.

## 3. First Launch

On first launch, TalyerPOS creates a local SQLite database automatically.

Default database behavior:

```txt
<Electron userData>/talyer-pos.sqlite
```

This means each Windows user profile has its own database unless a custom path is configured.

## 4. Fresh Start Login

Use the default Owner account only for first setup:

```txt
Username: owner
Password: 0000
```

Use Super Admin for system setup:

```txt
Username: superadmin
Password: superadmin123
```

Important:

- change default passwords before real operations
- create named accounts for real users
- disable demo accounts if they are not needed

## 5. Initial Store Setup Checklist

After first login:

1. Open `Settings`.
2. Set business name, address, contact number, email, and tax ID.
3. Upload or configure logo if available.
4. Configure receipt size and receipt output mode.
5. Add payment methods.
6. Add inventory categories.
7. Add suppliers.
8. Add services and labor cost.
9. Add inventory items.
10. Add mechanics.
11. Configure payroll settings if payroll is enabled.
12. Print mechanic QR ID cards.
13. Configure automatic backup in Super Admin.
14. Create a manual backup after setup.

## 6. Receipt Printer Setup

In `Settings`:

1. Select receipt output mode.
2. Choose printer if using printer output.
3. Save printer settings.
4. Print a test receipt through a sample sale or job receipt.

If the shop does not use a receipt printer, use PDF mode.

## 7. QR Scanner Setup

TalyerPOS assumes keyboard-wedge QR scanners.

Setup:

1. Plug in the USB scanner or pair Bluetooth scanner.
2. Open any text field in Windows Notepad.
3. Scan a mechanic QR code.
4. Confirm the scanner types the code and sends Enter.
5. Open TalyerPOS.
6. Scan mechanic QR from anywhere after login.

Scanner requirements:

- must output text as keyboard input
- should send Enter after each scan
- should scan the exact mechanic code/QR value

No camera or browser scanner setup is required.

## 8. Fresh Operational Reset

Use Super Admin reset only when the store wants a clean operational database.

Before reset:

- create a manual backup
- export database if needed
- confirm store owner approval

After reset:

- default Owner account is recreated
- operational data is cleared
- setup steps must be repeated
